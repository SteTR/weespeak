# WeeSpeak

Live speech recognition for you and your friends!

## WeeSpeak Part 1

### User Stories
- [X] User can pick a language
- [X] User can see their own words be recognized live
- [X] User can see their friend's words be recognized live

### User Stories

- [ ] User can logout
